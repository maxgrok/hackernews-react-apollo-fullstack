## Fullstack HackerNews Clone

Built with Apollo, React, GraphQL, and Node.js. 

## Run locally by ...

Run 'yarn dev' within 'servers' directory, then run 'yarn start' within the root directory of the project. Then navigate to 'localhost:3001'
